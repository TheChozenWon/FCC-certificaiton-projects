CSS Introduction

CSS (cascading style sheets) are used to add style to a webpage. CSS controls how HTML elements are displayed on screens.

The syntax of CSS uses a selector then {} with two declaration where each declaration has a property and a value.

h1 {color:blue; font-size:12px;}
Basic CSS

It's important to know the CSS box model such as its margins, padding, border and content

Other basics of CSS includes rgb, can edit colors of text and where to align text.
